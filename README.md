# Recipe Sharing Platform (Flask)

A full-featured web application built with Flask, where users can share recipes and chat with each other in real time.

## Features

- **User Management:** Registration, login, and OAuth-based authentication with third-party accounts
- **Real-Time Chat:** WebSocket-based live messaging system
- **Recipe Management:** Adding, deleting, and listing recipes
- **Interaction:** Commenting on recipes
- **Profile Management:** Full CRUD (create, read, update, delete) operations on the user profile

## Tech Stack

- **Backend:** Python, Flask
- **Real-Time Communication:** WebSocket
- **Authentication:** OAuth

## Setup

```bash
git clone https://github.com/GuvenEminSamil/flask-recipe-app.git
cd flask-recipe-app
pip install -r requirements.txt
python app.py
```

## Note

This project was built alongside a Spring Boot version of the same application, as an exercise in implementing the same functionality across different technology stacks. Check out the [Spring Boot version here](https://github.com/GuvenEminSamil/springboot-recipe-app).
